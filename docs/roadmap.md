# Roadmap

Quorum should grow from a local verifier into an evidence layer that can sit in
front of enterprise agent workflows.

## Now

- Keep the CLI reliable and easy to run.
- Improve claim extraction without hiding uncertainty.
- Make reports useful for a human reviewer.

## Next

- Accept PDFs and exported knowledge base pages.
- Add configurable source trust levels.
- Import reviewer decisions from filled claim-level exports.
- Support batch verification for multiple answers.

## Later

- Add an API service for agent integrations.
- Add a dashboard for review queues and audit history.
- Integrate with ticketing, HR, and support platforms.
- Add evaluation datasets for different business domains.
- Measure false positives, false negatives, and reviewer time saved.
